## 📌 About This Library

This is a personal prompt library I've built through hands-on work with Claude, n8n, Microsoft Copilot, SAP, and Power BI. Every prompt here has been used in a real project, client session, or workflow — and produced a high-quality output.

Each prompt includes:
- 📝 **The prompt** — copy-paste ready, with placeholders
- 🎯 **When to use it** — context and trigger conditions
- 🏷️ **Tags** — for quick filtering

## 🚀 How to Use

1. Browse the index above or navigate to a category folder
2. Find a prompt that fits your use case
3. Copy the prompt block
4. Fill in the `[bracketed placeholders]` with your specifics
5. Paste into Claude (or any LLM) and run

> **Tip:** The more context you add to the placeholders, the better the output. Don't leave them vague.
